## About

This repository contains the source code for **motioncard**, a web app built with [Next.js](https://nextjs.org/) and [Remotion](https://www.remotion.dev/).

## Setup

This is a [Next.js](https://nextjs.org/) project. If you're unfamiliar with it, check out the [Next.js Documentation](https://nextjs.org/docs).

To get started, clone the repo, install dependencies, and run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
